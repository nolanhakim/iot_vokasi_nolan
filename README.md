# Laporan Praktikum Internet of Things (IoT)

Repositori ini berisi laporan praktikum Internet of Things (IoT) yang mencakup berbagai simulasi dan eksperimen menggunakan ESP32, Wokwi, dan PlatformIO.

## 🛠 Teknologi yang Digunakan
- **ESP32** sebagai mikrokontroler utama.
- **Wokwi** sebagai simulator hardware.
- **PlatformIO** di Visual Studio Code untuk pengembangan kode.
- **C++** sebagai bahasa pemrograman.



## 🚀 **Cara Menggunakan**
1. **Clone repositori ini** ke komputer:
   ```sh
   git clone https://github.com/nolanhakim/iot_vokasi_nolan.git
   ```
2. **Buka folder proyek** dan jalankan kode di **Wokwi** atau **PlatformIO**.
3. **Eksperimen dan analisis hasil simulasi** sesuai laporan yang tersedia.
